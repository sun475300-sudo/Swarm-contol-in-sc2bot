## 교훈

### 인덴테이션 관리
1. **일관성**: 모든 Python 파일에서 4칸 스페이스 사용
2. **검증**: 배포 전 `python -m py_compile` 실행
3. **편집기 설정**: 탭 비활성화, 자동 들여쓰기 정리 활성화
4. **코드 리뷰**: 복사-붙여넣기 후 들여쓰기 확인

### 자동 검사 도구 활용
```bash
# 문법 검사
python -m py_compile 파일명.py

# 모든 Python 파일 검사
find . -name "*.py" -exec python -m py_compile {} \;
```

## 참고

- PEP 8 권장: 4칸 스페이스 사용
- 들여쓰기 혼합 금지: 탭과 스페이스를 함께 사용하지 않음
