## 수정 내용 상세

### 타입 힌트 개선
```python
# 이전
def run_command(cmd: list, ...):

# 수정 후
def run_command(cmd: List[str], ...):
```

### 인덴테이션 수정
```python
# 이전 (잘못된 인덴테이션)
def __init__(self, data_dir: str = "data/build_orders"):
    """
 Args:
 data_dir: Directory containing build order JSON files
     """
 self.data_dir = Path(data_dir)

# 수정 후 (올바른 인덴테이션)
def __init__(self, data_dir: str = "data/build_orders"):
    """
    Args:
        data_dir: Directory containing build order JSON files
    """
    self.data_dir = Path(data_dir)
```
