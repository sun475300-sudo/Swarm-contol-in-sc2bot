## 땅굴망이란?

**Nydus Network**는 저그의 지하 이동 통로 시스템입니다.

### 구성 요소
1. **Nydus Network** (본체)
   - 비용: 150 미네랄, 150 가스
   - 건설 시간: 50초
   - 요구사항: Lair (레어)

2. **Nydus Worm** (출구)
   - 비용: 75 미네랄, 75 가스
   - 건설 시간: 14초
   - Network에서 건설

### 작동 원리
```
[Network] ← 유닛 탑승
    ↓
(지하 이동)
    ↓
[Worm] → 유닛 하차 → 공격!
```

---

## 건설 순서

### 1. 레어 건설 (필수)
```
Hatchery → Lair
비용: 150 미네랄, 100 가스
시간: 80초
```

### 2. Nydus Network 건설
```python
타이밍: 4분 (240초)
위치: 본진 근처
비용: 150/150
```

### 3. Nydus Worm 건설
```python
타이밍: Network 완성 후
위치: 적 기지 후방 (아래 참고)
비용: 75/75
```

### 4. 유닛 투입
```python
유닛 선택 → Network 클릭 → 탑승
자동으로 Worm으로 이동
```

## 전략별 사용법

### 🎯 전략 1: Main Attack (주공격)
**시기**: 5분 이전
**목적**: 적 본진 파괴

#### Worm 위치
```
적 본진 후방 (15거리)
  ┌─────────┐
  │ Enemy   │
  │  Base   │
  └─────────┘
       ↓ (맵 중앙 반대편)
     [Worm]
```

#### 투입 병력
- Queen 3마리 (수혈용)
- Ravager 4마리 (담즙)
- Roach 8마리 (주력)
- **총 15유닛**

#### 공격 순서
1. 일꾼 학살 (Probe, SCV)
2. 건물 파괴 (Pylon, Supply Depot)
3. 군대 교전

---

### 🏗️ 전략 2: Multi Harass (멀티 견제)
**시기**: 5~10분
**목적**: 적 확장 기지 방해

#### Worm 위치
```
적 2번째 멀티
  [Main]
     ↓
  [Natural] ← 여기가 아니라
     ↓
  [3rd] ← 여기!
   ↑
 [Worm]
```

#### 투입 병력
- Zergling 10마리 (빠른 학살)
- Roach 5마리 (건물 파괴)

#### 전술
1. Worm 건설 → 적 발견 전
2. 일꾼 전멸 시키기
3. 멀티 건설 방해
4. 적 오면 Worm으로 철수

---

### 🚪 전략 3: Backdoor (백도어)
**시기**: 10분 이후
**목적**: 측면 공격

#### Worm 위치
```
     [우리]
        ↓
   [중앙 전선]
        ↓
측면 → [Worm] → [적 본진]
```

#### 투입 병력
- Hydralisk 8마리 (DPS)
- Roach 7마리 (탱킹)

#### 전술
1. 정면에서 견제
2. 측면 Worm으로 병력 투입
3. 적이 돌아오면 정면 압박
4. 협공으로 승리

---

## 최적 위치 선정

### ✅ 좋은 위치
1. **적 시야 밖**
   - 건설 중 발각 안 됨
   - 완성 후 서프라이즈

2. **건물 뒤**
   - 파일런 뒤
   - 막사 뒤
   - 적이 못 봄

3. **언덕 위**
   - 시야 차단
   - 방어 어려움

4. **적 멀티 근처**
   - 일꾼 바로 공격
   - 경제 타격

### ❌ 나쁜 위치
1. **적 정면**
   - 바로 발각
   - 즉시 파괴됨

2. **적 군대 옆**
   - 건설 중 공격 받음

3. **좁은 길목**
   - 유닛 나오기 어려움

4. **아군 기지**
   - 의미 없음

## 투입 병력 우선순위

### 우선순위 순서
```python
1위: Queen (3마리)
  → 수혈로 아군 치료

2위: Ravager (4마리)
  → 담즙로 건물 파괴

3위: Roach (8마리)
  → 주력 공격

4위: Hydralisk (5마리)
  → 높은 DPS

5위: Zergling (10마리)
  → 일꾼 학살용
```

### 병력 구성 예시

#### 올인 공격
```
Queen:    3
Ravager:  6
Roach:   15
합계:    24 유닛
```

#### 견제
```
Zergling: 20
Roach:     5
합계:     25 유닛
```

#### 백도어
```
Hydralisk: 10
Roach:     10
Queen:      2
합계:      22 유닛
```

## 실전 팁

### 💡 Tip 1: 멀티 Worm 전략
```python
# 여러 곳에 Worm 건설
Worm 1: 적 본진
Worm 2: 적 멀티
Worm 3: 측면

# 병력 분산 투입
→ 적이 대응 못 함
```

### 💡 Tip 2: 탈출 경로
```python
# 본진에도 Worm 건설
적 공격 오면:
  병력 → Network 탑승
  → 본진 Worm으로 즉시 귀환
```

### 💡 Tip 3: 전선 유지
```python
# 병력 수송로
후방 생산 → Network
→ 전선 Worm → 즉시 교전
```

### 💡 Tip 4: 자원 확보
```python
# 중립 확장 빠르게 먹기
Worm 건설 → 중립 멀티
→ Drone 투입
→ 해처리 건설
```

### 💡 Tip 5: Worm 방어
```python
# Worm 주변에 병력 배치
Spine Crawler 2개
Spore Crawler 2개
→ Worm 파괴 방지
```

## 🎮 봇의 자동 땅굴망 사용

### NydusNetworkTrainer가 자동으로:

1. **최적 위치 계산**
   - 게임 상황 분석
   - 적 위치 파악
   - 전략 결정

2. **병력 자동 투입**
   - 우선순위대로 탑승
   - 최대 15유닛씩
   - 지속적 증원

3. **자동 공격 명령**
   - Worm에서 나오면 즉시 공격
   - 타겟 우선순위:
     1. 일꾼
     2. 건물
     3. 군대

4. **재건설**
   - Worm 파괴 시 자동 재건설
   - 60초 쿨다운

## ⚠️ 주의사항

### 실패 요인
1. **너무 늦게 사용**
   - 4분에 건설 시작
   - 5분에 공격 시작

2. **병력 부족**
   - 최소 15유닛 필요
   - 8유닛 미만은 실패

3. **위치 실수**
   - 적 시야에서 건설
   - 즉시 파괴됨

4. **공격 안 함**
   - 유닛 투입만 하고 방치
   - 봇은 자동 공격

### 해결책
```python
✅ 4분에 정확히 건설
✅ 15유닛 준비 후 투입
✅ 적 시야 밖 위치 선택
✅ NydusNetworkTrainer 사용 (자동 공격)
```
